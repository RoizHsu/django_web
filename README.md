## 安裝與執行



```bash
# 克隆專案
git clone https://github.com/RoizHsu/django_web.git
cd django_web

# 安裝依賴
pip install -r requirements.txt

# 執行資料庫遷移
python manage.py migrate

# 啟動開發伺服器
python manage.py runserver
```
